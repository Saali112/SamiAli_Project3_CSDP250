Project: Binary Search Tree in C++
Files:
- binarySTree.h: Contains the structure and declarations for the Binary Search Tree.
- binarySTree.cpp: Implements the Binary Search Tree functions.
- main.cpp: The main program to demonstrate the functionality.
- readme.txt: This file.

Description:
This project generates 35 random numbers and organizes them into a Binary Search Tree. 
It provides traversal results in pre-order, in-order, and post-order.

Compilation:
Use the following command to compile:
g++ main.cpp binarySTree.cpp -o bst

Execution:
Run the program using:
./bst
